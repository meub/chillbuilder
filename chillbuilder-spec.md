## Chillbuilder

This is a spec for a new project called Chillbuilder.

The goal of this project is to build a web-based character creator and character stats manager tool, very similar to https://pathbuilder2e.com/ but instead of being for the Pathfinder 2e RPG system, this is for the Chill 2nd Edition system. The UI should be similar to pathbuilder (screenshot is included in this folder).

The app should be purely client-side and rely on local storage. It should have import/export capabilities.

The rules documents for chill are included as PDFs in this folder.

Link to more PDFs: https://files.spawningpool.net/docs/Vault2.0.-.TTRPG-Gamebooks/Chill/2nd%20Edition%20%28Mayfair%20Games%29/
